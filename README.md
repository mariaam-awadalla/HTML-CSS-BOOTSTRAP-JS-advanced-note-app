# 📝 Note App

A simple and interactive Note Taking Web Application built using **HTML, CSS, JavaScript, and Bootstrap**.
This app allows users to create, manage, and organize their notes visually appealing.

---

## 🚀 Features

* ➕ Add new notes instantly
* ✏️ Edit note title and content
* 🗑️ Delete notes 
* 📌 Pin important notes to the top
* 🎨 Change note Color Picker
* 🔍 Search notes by title or content
* 🌙 Dark Mode toggle
* 💾 Save notes using Local Storage (data persists after refresh)
* 📤 Export notes as JSON file

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5

## ⚙️ How It Works

* Notes are stored in the browser using **localStorage**
* Each note contains:

  * id
  * title
  * content
  * color
  * pinned status
* Notes are dynamically rendered on the page
* Pinned notes are always displayed at the top
